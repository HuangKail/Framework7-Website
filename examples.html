可以直接使用的一些案例

下面这些例子可以当做你的初始化模板来用。

你可以在 github 上下载所有这些案例。

单视图应用

这是最常见的一种布局，在一个视图中包含了所有的页面、工具栏、导航栏等。

新窗口打开

Tab Bar 应用

这是一个tab bar应用的例子。每一个 Tab 都包含了一个有独立浏览历史和页面跳转的View。

每一个 tab/view 都可以有不同的布局，不同的navbar类型。

新窗口打开

Split View Application

一般会在iPad上用到这种布局。特点是这个应用同时显示了多个View。

每一个View都是独立的，都可以有自己不同的布局和导航栏等。

新窗口打开

响应式的Split View Application

这是一个响应式布局的例子，左边的view在窄屏设备上会隐藏起来，就像iOS7上的邮件应用一样。

点击"新窗口打开"，然后调整你的浏览器宽度可以看到布局的变化。
